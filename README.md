# Rankine Cycle Analysis (Thermodynamics)

## Overview
Analyzed a regenerative-reheat Rankine cycle to evaluate thermal efficiency improvements using EES.

## Objectives
- Study effect of reheating and regeneration
- Analyze thermodynamic performance
- Calculate efficiency improvements

## Tools Used
- EES (Engineering Equation Solver)

## Key Results
- Efficiency improved with reheating and regeneration
- T-s diagram used to visualize cycle performance

## Files Included
- Project report
- Presentation slides
- EES simulation file

## Learning Outcomes
- Strong understanding of power cycles
- Practical use of thermodynamic equations
- Visualization of cycles using T-s diagrams
